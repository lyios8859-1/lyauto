# 手把手完成一个简单的前端的脚手架

默认您已经配置好开发环境

## 初始化 npm

```shell
  npm init [-y]
```

## 所需工具库

```shell
  npm i -S chalk cli-table commander inquirer ora [--registry=https://registry.npm.taobao.org]
```

## 目录结构

```
.
├── bin
├── commands
├── images
├── node_modules
├── package.json
├── README.md
└── templates
```
